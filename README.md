# Aadhaar Registration Form

A simple Aadhaar registration form built using HTML and CSS.

## Features
- Input fields for full name, date of birth, and address
- Clean and simple user interface
- Responsive design using pure CSS

## Tech Stack / Skills Used
- HTML5
- CSS3
