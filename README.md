# Aadhaar Registration Form

A complete Aadhaar registration system built using HTML, CSS, Python
with a database backend for storing registration data.

## Features
- Input fields for full name, date of birth, and address
- Form validation to ensure correct data entry
- Python backend for processing form data
- Database integration for storing Aadhaar records
- JSON for data handling and configuration
- Clean and simple user interface

## Skills Used
- HTML5
- CSS3
- Python
- Form Validation
- Database Management (DB)
- JSON Data Handling
- Git & GitHub

## Tech Stack
| File | Technology | Purpose |
|------|-----------|---------|
| index.html | HTML5 | Structure of the form |
| style.css | CSS3 | Styling and layout |
| app.py | Python | Backend processing |
| database.db | SQLite | Storing registration data |
| data.json | JSON | Data handling & configuration |

## How to Run

### Frontend
1. Clone the repository
git clone https://github.com/Gaganshekar/aadhaar-registration.git

2. Open `index.html` in your browser
### Backend
1. Install Python dependencies
2. Run the Python file
